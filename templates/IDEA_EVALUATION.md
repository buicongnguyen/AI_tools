# Evidence-Based Idea Evaluation

Evaluate: **[IDEA]**

As of: **[DATE]**

Target market/geography: **[MARKET]**

## Instructions

1. Restate the idea, user, painful job, and proposed value.
2. Search the web and prefer primary sources.
3. Build a competitor/substitute table with direct links and dates.
4. Separate verified facts, inferences, estimates, and opinions.
5. Identify the five riskiest assumptions.
6. Design the cheapest experiment to falsify each assumption.
7. Score the idea using the rubric below.
8. Give Proceed, Experiment, or Pause; state what evidence would change the decision.

## Rubric

| Dimension | Weight | Score | Evidence |
|---|---:|---:|---|
| Problem severity/frequency | 20 | | |
| Demand evidence | 20 | | |
| Differentiation | 15 | | |
| Distribution | 15 | | |
| Technical feasibility | 10 | | |
| Economics | 10 | | |
| Risk/compliance | 10 | | |

## Required output

- Executive decision.
- Claim-evidence table.
- Competitors and substitutes.
- Assumption/risk register.
- 2-week validation plan.
- Prototype acceptance criteria.
- Sources with publication dates and verification date.
